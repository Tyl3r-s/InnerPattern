# InnerPattern

## Purpose
A MERN stack single-page application where users can track their thoughts and 'moods' through journal entries.

## Built With
* Node
* Express
* React
* GraphQL
* MongoDB
* Express-Session
* Bootstrap
* Mongoose
* Apollo
* Bcrypt
* JWT
